# ERP SaaS React + TypeScript

Proyecto base para un ERP SaaS multi-inquilino.

## Estructura

- **src/components**: Componentes reutilizables de la UI.
- **src/pages**: Páginas principales de la aplicación.
- **src/services**: Lógica de acceso a datos o llamadas a APIs.
- **src/types**: Definición de tipos y modelos TypeScript.
- **src/hooks**: Hooks personalizados para compartir lógica.

## Scripts disponibles

- `npm start` - Levanta el servidor de desarrollo.
- `npm run build` - Genera el build de producción.
- `npm test` - Ejecuta pruebas (no configuradas por defecto).

Este repositorio contiene la configuración mínima con webpack y TypeScript para comenzar un proyecto React.
